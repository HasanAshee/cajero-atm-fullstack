# 🏧 ATM Simulator — Frontend

A full-stack ATM web application built with Angular and Node.js, simulating real banking operations with a focus on security and user experience.

🔗 **Live Demo:** [cajero-atm.netlify.app](https://cajero-atm.netlify.app)
🔗 **Backend Repo:** [cajero-atm-fullstack](https://github.com/HasanAshee/cajero-atm-fullstack)

## ✨ Features
- Secure PIN authentication with bcrypt hashing
- Full CRUD operations: deposits, withdrawals, balance checks
- Complete transaction history per user
- JWT-based session management
- Responsive UI built with Angular

## 🛠 Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| Angular 17 | Node.js + Express | MongoDB |
| TypeScript | JWT Auth | Mongoose |
| RxJS | bcrypt | |

## 🚀 Getting Started
```bash
npm install
ng serve
```
Navigate to `http://localhost:4200`
